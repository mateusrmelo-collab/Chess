# ♟️ Chess

Um jogo de xadrez desenvolvido com **HTML, CSS e JavaScript puro**, criado com o objetivo de estudar lógica de programação, manipulação do DOM e implementação das regras oficiais do xadrez sem o uso de bibliotecas ou frameworks.

O projeto foi desenvolvido do zero, implementando manualmente todas as mecânicas do jogo, incluindo uma IA para partidas contra o computador, além de uma interface moderna inspirada em plataformas como o Chess.com.

---

## ✨ Funcionalidades

* ♟️ Tabuleiro interativo 8×8
* ♔ Movimentos completos de todas as peças
* 👑 Promoção de peões
* 🏰 Roque (curto e longo)
* ⚔️ En Passant
* 🎯 Captura de peças
* 🔄 Alternância automática de turnos
* 👀 Detecção de xeque
* ♚ Detecção de xeque-mate
* 🤝 Detecção de afogamento (Stalemate)
* 🤖 Modo contra IA
* 📜 Histórico de jogadas
* 📝 Notação das jogadas
* 💡 Destaque para:

  * peça selecionada;
  * movimentos legais;
  * capturas possíveis;
  * último movimento realizado;
  * rei em xeque.
* 🎨 Interface inspirada no Chess.com
* 📱 Layout responsivo

## 🤖 Inteligência Artificial

O projeto conta com um modo de jogo contra o computador, desenvolvido sem o uso de bibliotecas externas.

A IA utiliza:

* Algoritmo **Minimax**
* **Poda Alfa-Beta** para otimização da busca
* Avaliação baseada no valor das peças
* Tabelas posicionais (*Piece-Square Tables*)
* Diferentes níveis de profundidade de busca

Isso permite que a IA realize jogadas estratégicas mantendo um bom desempenho mesmo sendo implementada inteiramente em JavaScript.
