# ♟️ Chess

Um jogo de xadrez desenvolvido com **HTML, CSS e JavaScript puro**, criado com o objetivo de estudar lógica de programação, manipulação do DOM e implementação das regras oficiais do xadrez sem o uso de bibliotecas ou frameworks.

O projeto foi desenvolvido do zero, implementando manualmente as mecânicas do jogo e buscando uma interface moderna inspirada em plataformas como o Chess.com.

---

## ✨ Funcionalidades

* ♟️ Tabuleiro interativo 8×8
* ♔ Movimentos completos de todas as peças
* 👑 Promoção de peões
* 🏰 Roque (curto e longo)
* ⚔️ En Passant
* 🎯 Captura de peças
* 🔄 Alternância automática de turnos
* 👀 Detecção de xeque
* 📜 Histórico de jogadas
* 📝 Notação das jogadas
* 💡 Destaque para:

  * peça selecionada;
  * movimentos legais;
  * capturas possíveis;
  * último movimento realizado;
  * rei em xeque.
* 🎨 Interface inspirada no Chess.com
* 📱 Layout responsivo
